## 🛠️ COMMANDES DISPONIBLES

### Développement
```bash
# Démarrer le serveur (déjà lancé)
npm run dev

# Build production
npm run build

# Serveur production
npm run start

# Vérifier le code
npm run lint
```

### Base de Données (à configurer)
```bash
# Générer client Prisma
npm run db:generate

# Appliquer schéma
npm run db:push

# Charger données
npm run db:seed

# Interface graphique
npm run db:studio
```
